More Script

Syntax

More string

Synopsis

Paginates string to stdout.

Detailed Description

Paginates string to stdout.  After each page of output, output pauses until you hit a return.  This can be used to paginate script output e.g in the example below:

More [SpecList]

The output of the SpecList script (formatted list of spectra) is paginated.
