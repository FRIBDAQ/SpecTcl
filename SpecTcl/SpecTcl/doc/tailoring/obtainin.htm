Obtaining Skeleton Software

SpecTcl Home  General Information User Guide Programmer's Guide Obtaining and Installing

The skeleton software is a set of files which you can modify to easily adapt SpecTcl to your event data. It includes two C++ source files and a Makefile:

  • Makefile - builds your customized version of SpecTcl
  • AppInit.cpp - Initializes SpecTcl. It will only need to be modified if you must adapt SpecTcl to a non NSCL buffer source.
  • UserCode.cpp - Contains a sample Event Unpacker.

For most people using SpecTcl with NSCL data buffers, only UserCode.cpp will need to be modified. To obtain the skeleton software, set your default directory to the place you want to build SpecTcl and:

cp $INSTBASE/Skel/* .

In this line $INSTBASE is assumed to be the base of the directory tree into which SpecTcl was installed (e.g. /usr/opt/SpecTcl for Linux or /usr/local/SpecTcl for Compaq Tru-64 Unix).
