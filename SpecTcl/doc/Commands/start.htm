start - SpecTcl Command

SpecTcl Home  General Information User Guide Programmer's Guide Obtaining and Installing

Syntax

start

Synopsis

Starts data analysis. Data analysis can then be halted via the stop command.

Example

# The following multiline example starts data analysis and
# processes data for 10 seconds or until an end of file is encountered
# on the input 
#
start
after 10000 stop
