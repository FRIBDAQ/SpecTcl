clear - SpecTcl Command

SpecTcl Home  General Information User Guide Programmer's Guide Obtaining and Installing

Syntax

clear -all
clear pattern   
clear -id id [id...]

Synopsis

Zeroes the channesl of selected or all spectra.

  • Returns a list of the set of spectra which didn't exist and therefore could not be cleared.

Detailed Description

Spectra can be referred to either by id or by name. Spectra can be selectively cleared or all spectra can be cleared.

clear -all

Clears all spectra known to SpecTcl.

clear pattern

Clears all of the spectra whose name matches pattern.

clear -id id [id...]

Clears spectra by id.

Examples

clear -all ;# all spectra are cleared.
clear spec1 spec3 ;# Clears spectra named spec1 and spec3
clear -id 1 5 7 ;# Clears spectra with id's 1, 5 ,and 7.
for { set i = 0} {$i <= 10} {incr i 1} { clear -id $i } ;# clears spectra with ids 0 through 10.
