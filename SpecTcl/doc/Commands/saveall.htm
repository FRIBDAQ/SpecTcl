saveall script

Syntax

saveall filepath

Detailed Description

Saves the analysis state of SpecTcl to the file filepath. The analysis state of SpecTcl is defined currently as:

  • The spectra which are defined but do not depend on deleted parameters.
  • The Gates which are defined but don't depend on deleted parameters.
  • The applications of the gates to spectra for which undeleted gates exist.
