Tailoring SpecTcl for an experiment

Overview

This page points you to documentation which describes how to tailor SpecTcl to handle the data from an experiment. Included is documentation wich:

  • Describes the way SpecTcl takes a block of experimental data and transforms it into a CEvent class.
  • Describes how to get the skeleton software you will need to modify from the SpecTcl installation.
  • Describes how to modify skeleton software you have copied.
  • Describes how to use the TreeParameter subsystem to structure your unpacked data.
  • Describes how to build a version of SpecTcl tailored to your needs
  • Describes how to make advanced modifications to SpecTcl's skeleton to support analysis of non NSCL data.
  • Describes how to add commands to a tailored version of SpecTcl.
  • Describes how to access Tcl Variables from within your code.
