What is SpecTcl?

SpecTcl (pronounced Spectacle) is a powerful nuclear event data analysis too.  It provides an object oriented C++ framework for histogramming and other data analysis operations.  The Tcl/TK scripting language is embedded as the program's command language, providing the user with a powerful, extensible, command set as well as the ability to build custom graphical user interfaces or extend existing ones. The Xamine display program provides SpecTcl with a powerful visualization component.

SpecTcl is known to run on the following operating systems:

  • Compaq Tru64 Unix
  • Linux
  • HP-UX

SpecTcl requires the following software:

  • Tcl/Tk 8.0 or higher.
  • Bison/Flex
  • Motif or Lesstif.
  • g++
