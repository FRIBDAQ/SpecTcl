Syntax

    specstats ?pattern?

Synopsis

Provides information about the spectrum under and overflows for spectra.

Detailed Description

If pattern is provided information is restricted to spectra whose name match that pattern, which can include glob wildcard characters. If pattern is not present on the command line, information is provided for all spectra (as if the pattern were *).

The command returns a list of Tcl dicts. Each dict contains overflow and underflow statistics for a single spectrum.

By overflow we mean the number of times a parameter has been too large to increment the spectrum. Similarly by underflow we mean the number of times a parameter has been too small to increment a spectrum.

Each dict has the following keys:

Key Value
nameName of the spectrum.
underflows Either the number of underflows or a 2 element list containing the number of x and y underflows
overflowsEither the number of overflows or a 2 element list containing the number of x and y overflows

Note that if you are using the Xamine displayer the Info button will provide you with the over/underflow data. If you use the root displayer, this information will be displayed in the title box on the spectrum.
