Syntax

    scontents ?-json? spectrum-name

Synopsis

Returns spectrum contents for spectrum-name. If -json is specified the data are encoded using JSON otherwise the data are in a simple Tcl list format.

Detailed Description

Returns the contents of the specified spectrum.

If the optional -json switch is specified, the returned string is a JSON encoded array where each array element is an object with the following attributes:

Field NameField Contents
xchanAn x axis channel number
ychanOnly present in 2-d spectra, a y axis channel number
valueThe number of counts in the specified channel.

Note that minimal compression is obtained by only including channels with non-zero values in the array.

If the optional -json switch is not provided, the returned string is in a format that is easy for Tcl scripts to manipulate. In this case, the returned string is just a list of two (1-d spectra) or three (2-d spectra) sublists. The last element of each list is the number of counts in a channel while first elements specify the x and optionally (2-d spectra) y channel number.
