ApList Script

Syntax

AppList [applicationlist] [format]

Synopsis

Produces a nicely formatted list of the gate applications.

Detailed Description

Produces a nicely formatted list of the gate applications where:

  • applicationlist is the output of one of the apply -list commands.  If omitted, this defaults to: [apply -list] which produces a list of all gate applications.
  • format if supplied can be one of the following formatting control options:

    a*lign    - Lists description horizontally aligning overflow fields with columns.
    d*own  - Stacks separate descdription sections downward.
    c*down- (default) Stacks downward if the gate is a band or countour.
