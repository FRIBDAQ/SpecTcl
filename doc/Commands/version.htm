Syntax

    version

Synopsis

Returns the SpecTcl version string. This is normally a string of the form mm.nn-xyz where mm is the major version of the program, nn the minor version and xyz an edit level.

xyz can also indicate a pre-release or release candidate version by being either beta for a pre release or rcnn for a release candidate.

Example:

%  version
3.4-005

The intent of this command is to allow SpecTcl scripts to be able to know which SpecTcl commands have been implemented in the vesion of SpecTcl under which they are running.
