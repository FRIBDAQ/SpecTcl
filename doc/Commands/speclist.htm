SpecList script

Syntax

SpecList [spectrum-definition-list] [format]

Synopsis

Produces a nicely formatted list of spectra.

Detailed Description

Produces a nicely formatted list of spectra where:

  • spectrum-definition-list - is the output of one of the SpecTcl spectrum -list commands.  If omitted this defaults to the output of spectrum -list.
  • format                            - controls the format of the output:

down - stacks parameters downwards
wide    - stacks parameters downward double wide.
long     - stacks parameters horizontally, sometimes running to the next line.
