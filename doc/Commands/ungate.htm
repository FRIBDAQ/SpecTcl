ungate -remove condition from spectrum

 

Syntax

ungate Spectrum1 [Spectrum2 ...]

Synopsis

Removes the gate condition on the set of spectra specified.

Detailed Description

For each spectrum in the list, the gate on that spectrum is replaced by the -TRUE- (always true) gate.  This effectively removes any gate condition on the increment of the histogram.

See also:

  • gate - to create, list and delete gates.
  • apply - to apply a gate to spectra.

Examples

ungate  Spectrum1     ;# Remove gate from Spectrum1
ungate Spectrum2 Spectrum3 ;# Remove gates set on Spectrum2 and 3.
