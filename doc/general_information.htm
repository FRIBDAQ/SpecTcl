What is SpecTcl?

SpecTcl is the new NSCL histogramming and data analysis package. This program uses the following software technology:

  • Xamine displayer package.
  • Object oriented design and implementation with C++
  • Embedded Tcl/Tk command language to support flexible scripting and tailorable GUIs.
  • Standard Template Library (STL ) heavy use of container classes.
  • HTML on-line user guide.

Functional Information

The current version of SpecTcl is 0.1 This version is a pre-release which provides the following functionality:

  • Dynamically created spectra of any number
    1-d long spectra
    2-d word spectra
  • Spectra dynamically bound and unbound from Xamine
  • Data sources:
    File
    Tape
    Online NSCL unix data source.
  • Complex dynamically creatable gates.
  • Additional Spectrum types:
    1-d word
    2-d Byte
    2-d summary spectrum.
    Bit register spectrum.
  • Tcl Scriptable, dynamically created Pseudo parameters.
  • Platform independent byte ordering.
