Provide the Filter Gate

Each filter is associated with a gate. When the filter is enabled, only the events that satisfy the gate will be written to the filter file. This dialog prompts you for the filter's gate.

The top part of the dialog is a limited object browser that shows you the top level definitions of each gate known to SpecTcl. Double click on a gate to select it. The selected gate is displayed below the browser. If you select the wrong gate by mistake, simply double click the correct gate to replace youre selection.

The buttons at the bottom of the filter wizard have the following meanings:

  • Cancel Cancel the creation of the filter and dismiss the wizard.
  • Help Display this help text.
  • Back Go back to the previous step of the filter wizard (choosing a filter name).
  • Next Continue defining the filter.
