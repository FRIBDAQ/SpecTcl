The SpecTcl GUI Menu Bar

The SpecTcl GUI menu bar provides quick access to functions that may not be easily available within the section of the browser you are looking at. For example, if you are browsing spectra, and want to create a new gate, it may be easiest to do this by selecting that function from the Gate menu than by scrolling up to the Gates folder and right clicking it.

Additional functionality is available from some of the menu bar menus that is not available from the browser. For example, you can:

  • Save and restore the program state
  • Setup the SpecTcl data source
  • Setup and control filters
  • Read spectrum files
  • Access this help information
  • Access the SpecTcl online manual

The menus currently supported are:

  • File
  • Data Source
  • Filters
  • Spectra
  • Gate

In addition to these, the Help menu is available to bring up full help on the Gui and SpecTcl online documentation.
