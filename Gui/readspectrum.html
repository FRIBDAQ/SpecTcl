Restoring spectrum contents from file

This dialog consists of three vertically stacked sections. The top section is a file selector. Use it to choose the name of the file you are reading. The middle is a set of three checkoxes:

  • Snapshot If selected, the spectrum read from file is read in such a way that it will never be incremented.
  • Replace existing spectra If set, a spectrum that already exists will be ovewritten with the data read from file. If not, a unique spectrum name will be created based on the existing spectrum name.
  • Bind to Display If checked, the spectrum is bound to the display

At the bottom of the dialog, there are three buttons:

  • Ok Reads the spectrum file and dismisses the dialog
  • Cancel Dismisses the dialog and does nothing else.
  • Help Displays this help page
