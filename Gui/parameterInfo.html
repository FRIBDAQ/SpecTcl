Information available from parameters

SpecTcl has three fundamental types of parameters:

  • Tree parameters, which are declared in the user's code to produce the SpecTcl parameters from raw events
  • Parameters declared with the SpecTcl parameter command.
  • Pseudo parameters which are computed from other parameters via a Tcl script

Tree parameters display information about suggested axis ranges, binning and optionally units. Parameters declared with parameter do not have this additional information. Note that it is possible to convert a parameter declared with the parameter command into a Tree parameter by providing range and optionally unit information about the parameter.

Pseudo parameters display with an icon that looks like f(x) indicating they are computed. If a pseudo parameter depends on other parameters, it can be drilled into to inspect the parameters it depends on. At this time, you can only inspect the body of a pseudo parameter by right clicking it and selecting Pseudo... from the parameter context menu.
