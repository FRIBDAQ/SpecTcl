Deleting Multiple Gates

This dialog allows you to delete onee or more gates. The left side of the dialog is a limited object browser that allows you to view the top level information about the gates currently defined. The right side of the browser is a listbox that will contain the gates you intend to delete. The bottom of the browser has several buttons that control the action of the dialog

To select a gate for deletion, double click it in the browser. If, by mistake, you selected the wrong gate, double click it in the listbox to remove it from the list of gates to be deleted.

The buttons at the bottom of the widget have the following functions:

  • Ok After providing a confirmation prompt, deletes the gates that you have put into the gate list box. In either event, the dialog is dismissed.
  • Cancel Dismisses the dialog box without deleting any gates
  • Help Displays this help text
