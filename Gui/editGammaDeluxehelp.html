Gamma 2d Deluxe spectrum editor

A gamma 2d Deluxe spectrum is a gamma spectrum with an independent list of x and y parameters. Each pair of x/y parameters results in an increment. For example suppose you have on the X axis parameters a and b and on the Y axis parameters c and d. The spectrum will be incremented for all valid pairs: a vs. c, a vs. d, b vs c, and b vs d

To define a gamma spectrum use the browser window on the left to select parameters and the appropriate X-> and Y-> buttons to add a selected parameter to the list of parameters for a specific axis.

Remove a parameter from an axis list by double clicking it.

Use the entry boxes below each parameter list to set the axis range and number of channels. If you have not set these prior to accepting the first parameter, and the first parameter has range information defined, that range information, and your default 2-d channel counts will be used to fill in these entries.

Once you are done, click OK to define the spectrum and dismiss the editor, click Accept to define the spectrum and keep the dialog up so that you can define another spectrum, Cancel to dismiss the dialog without creating a spectrum and Help for this message.
