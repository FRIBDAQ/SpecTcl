Gate Menu

The gate menu allows you to manipulate SpecTcl gates. SpecTcl gates are used to conditionalize the increment of spectra to which they have been applied. Gates are also used to determine which events will be written to filtered data sets.



Top

Create

The Gate->Create... command prompts you graphically to create a new gate. Note that many simple gate types (slices, contours, bands and their gamma counterparts) can be entered graphically on the displayer window. This menu is best used to create compound gates (and, or, not for example).

The command prompts for a gate name and gate type. When the gate type has been selected, the appropriate gate editor is displayed to allow you to define the gate.



Top

Apply

The Gate->Apply... command allows you to select a gate to apply to several spectra. once applied, the spectra will only increment for events that satisfy the gate.



Top

Delete

The Gate->Delete... command allows you to delete a gate. In reality deleted gates are just turned into gates that are always false. This has some implications for gates that depend on the gate that has been deleted, but probably these are not as bad as other choices for what to do with a deleted gate.
