Spectra Menu

Save Contents of Spectra

Spectrum->Save Contents of Spectra... allows you to save the contents of one or more spectra to file. This is identical to the File->Save Contents of Spectra... command



Top

Read Spectrum File

Spectrum->Read Spectrum File... allows you to select a file written via the swrite command and read it back in. For more information, See File->Read Spectrum file...



Top

Clear All

The Spectrum->Clear All command clears all channels of all spectra. This is equivalent to the 

clear -all
Spectcl command.

Top

Create

The Spectra->Create... command allows you to create a spectrum. The dialog prompts for spectrum name and type. When the spectrum type is selected, an editor for the appropriate spectrum type is displayed. The spectrum can be created with a gate initially applied. It is also possible to create a set of spectra on an array of SpecTcl parameters for 1-d spectra.



Top

Delete

The Spectra->Delete... command graphically prompts for a set of spectra to delete. Once you have selected the spectra, you will be given a confirmation prompt before the spectra are actually deleted.



Top

Apply Gate

The Spectra->Apply Gate... command graphically prompts for a gate to be applied to a number of spectra.
