Selecting a Gate and Applying it To One or More Spectra

This dialog allows you to select a single gate and apply it to one or more spectra. The dialog consists of three sections:

  • A limited browser at the left side of the dialog that shows top level information about spectra and gates
  • An area at the right side of the dialog that shows you what you have selected. At the top of this area is the selected gate. At the bottom of this area, a list of spectra selected.
  • An area at the bottom of the dialog that contains buttons named:
    1. Ok when clicked this button applies the gate you have selected to the set of spectra in the listbox, and dismisses the dialog.
    2. Cancel when clicked, this button dismisses the dialog and does nothing
    3. Help When clicked, this buttons displays this help text

To select a gate to be applied, double click a gate from the Gates folder of the browser. If you have made an error and selected the wrong gate, just double click the correct one to replace your current selection.

To add Spectra to the list of spectra in the listbox at the right side of the dialog, double click the appropriate spectrum fromt he Spectra Folder of the browser in the dialog. If you make a mistake and add the wrong spectrum to the listbox, double click it in the listbox, to return it to the browser.

Note that this dialog is modal
