Save spectrum contents

This dialog allows you to save the contents of one or more spectra to file.

The top part of the dialog contains an object browser limited to spectra and a list of the spectra you have chosen to write. To add a spectrum to the list, double click it in the browser. To remove a spectrum from the list, double click it in the list labelled Spectra to Write

The bottom of the dialog contains the following buttons:

  • Ok Prompt for a filename in which to write the spectra
  • Cancel Dismiss the dialog without taking any action
  • Help Display this dialog
