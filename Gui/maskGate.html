Defining a bit mask gate

Bit mask gates require a parameter and a bitmask to complete their definition. The editor for bit mask gates consists of a limited object browser at the left of the editor. A status area at the right that displays the currently selected parameter and a bit mask editor at the bottom above the action buttons.

To select a parameter for the gate, double click it in the browser. If you have chosen the wrong parameter just double click the correct one.

The bit mask editor has the following elements:

  • A button labelled Clear All that will clear all of the mask bits
  • A button labelled Set All that will set all of the mask bits
  • Individual checkboxes for each bit in the mask
  • An entry for the numeric value of the mask which updates the bits of the mask when you hit either the enter or tab keyboard keys. The numeric value can be entered as:
    • A decimal number.
    • A hexadecimal number by preceding the digits with 0x
    • An octal number by preceding the digits with 0
    Note that the checkboxes that represent the current bitmask are groped into fours making hexadecimal the preferred radix.

The buttons at the bottom of the editor are:

  • Ok Click this to create the gate and dismiss the dialog.
  • Accept Click this to create the gate and leave the dialog up
  • Cancel Click this to dismiss the dialog without making a new gate
  • Help Clicking this displays this help message. If you have selected a different gate type, clicking help again will display the help for the specific gate editor you have selected.
