Using the object browser

It may be useful to look at the vocabulary page if you are uncertain what some of the terms in this page mean.

Sections of the browser window

The Browser window has three sections.

  • A menubar containing a set of pulldown menus from which major browser commands can be selected.
  • The object browser, which allows you to view and manipulate the SpecTcl Objects.
  • A status line which displays information about the analysis currently underway.

This page will describe the functionality available in the middle or object browser section.

The object browser consists of toplevel folders for each of the major object types supported by SpecTcl. These object types are Spectra, Parameters, Gates and Variables. Each of these Top level folders can be opened by clicking on the + to the left of the folder. Each toplevel folder also has a context menu that you can access by right clicking on the folder.

When you open a top level folder you will see objects and subfolders. Subfolders do not have context menus associated with them. Objects, however do have associated context menus that once more are accessed via a right click on the object. Subfolders can be opened by clicking on the + to the left of the folder. With the exception of variables, objects also have descriptions. Object description details can be accessed by clicking on the + to the left of the object name.

Each object also has a selection action associated with it. This action usually is to edit the definition or value of the object. The selection action is activated by double clicking the object.
