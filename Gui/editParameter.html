Editing a Parameter

This dialog allows yout to edit the value and unit of a parameter or, if the parameter you have chosen is an element of a treeparameter array, an entire array of parameters. You may also edit a parameter that is not a tree parameter. If you accept this edit, the parameter will be transformed into a tree parameter, and provide recommendations for axis definitions.

The top part of the dialog is a form that you can fill in to provide the suggested axis limits, binning and parameter units of measure. If the parameter is an element of an array, the entire array can be modified by checking the array checkbox.

A parameter is considered an array element if:

  • The last element of its name path is a number
  • There are other parameters with the same preceding path.

The buttons at the bottom of the dialog have the following functions:

  • Ok Accepts the modifications and dismisses the dialog
  • Accept Accepts the modifications but does not dismiss the dialog, so you can make further modifications
  • Revert Returns the top form to the settings the parameter currently has.
  • Help Displays this help text

Note This dialog is not modal.
