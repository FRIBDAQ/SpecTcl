2-d Spectrum editor

This is the 2-d spectrum editor. To define a 2-d spectrum, you must select an x and y parameter, describe the spectrum parameter axes, pick a spectrum name, and optionally select a gate to initially apply to the spectrum.

At the top of the dialog we have:

  • An entry labelled Spectrum Name:; Type the name of the spectrum you are creating here.
  • Some text that will display any gate you may want to apply to the spectrum. the spectrum specific editors will give you the opportunity to supply a gate
  • An array checkbox. If you build a 1-d spectrum from a treeparameter array element, you can create an entire array of spectra, one on each element of the array if this is checked

In the middle of the dialog we have a limited object browser that shows gates and parameters, and boxes that describe the x and y parameters currently chosen. One of the parameters is highlighted. This parameter is the one which will be selected next. To change the highlight, click on the name of the parameter you would like to highlight.

  • Double clicking a parameter in the browser selects it for the highlighted axis. The other axis then becomes the highlighted axis. If you have made an error and chosen the wrong parameter:
    1. Click the name of the parameter in the unhighlighted box to highlight it.
    2. double click the correct parameter in the browser
  • You can remove a parameter from a parameter box by double clicking the name
  • If a selected parameter is a tree parameter, suggested values for the axis limits and binning will be displayed in the entries. You may modify or set these values by typing them in.
  • To select a gate to be applied to the spectrum, double click a gate in the browser. If you selected the wrong gate, double click the correct gate. If you decide not to gate the spectrum, double click the gate name at the top of the dialog.

The buttons that are avaialable at the bottom of the dialog are:

  • Ok Creates the new spectrum as described and dismisses the dialog
  • Accept Creates the new spectrum but leaves the dialog up to allow you to create more spectra.
  • Cancel Cancels spectrum creation and dismisses the dialog.
  • Help Brings up this help dialog
If you attempt to create a spectrum that already exists, you will be prompted to confirm. If you accept, the existing spectrum will be deleted before the new one is created.
