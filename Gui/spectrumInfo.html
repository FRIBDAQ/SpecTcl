Information available from spectra

Opening a spectrum will reveal details about a spectrum that depend on the spectrum type. In general, all spectra will display the parameters and information about the range of parameter values covered by their axes and the number of bins on each axis.

1-d spectra

1-d spectra provide the following information on the same line of the browser as the spectrum object:

  • Spectrum type (1 and data type of channel e.g. long).
  • Gate that is applied (if any).
  • Range of parameter covered by the X-axis and number of bins on that axis
  • Where applicable the units of measure of the X-axis
  • If the spectrum is gated, the Gate column names the gate that is applied to the spectrum

In addition, drilling down into the spectrum will display the parameter that is histogrammed. In most cases, 1-d histograms are given names that match the names of their parameter.

2-d spectra

2-d spectra only show spectrum type and channel type information on the same line as the histogram. If the spectrum is gated, the name of the gate applied to the spectrum is listed in the Gate column. To see more information it is necessary to drill down into the spectrum. Drilling down into the spectrum provides the following information

  • X axis description. This includes the name of the parameter on that axis as well as axis range and binning and, where applicable, the units of measure of the underlying parameter.
  • Y axis description. Provides the same information for the Y axis as is provided for the X axis.

Summary spectra

Summary spectra provide the following information on the same line as the spectrum name.

  • Spectrum and channel type.
  • Y-axis range and binning
  • Where applicable, the Y axis units
  • Gate applied to the spectrum if any

Drilling down into the spectrum provides the following information:

  • X axis identification and range (given as [0, nparameters-1]). The X axis is identified as "X : parameters"
  • The Y axis identification range and units of measure taken from the first of the parameters
  • The list of parameters that make up channels on the X axis in the order in which they appear on that axis

1-d Gamma Spectra

1-d gamma spectra provide the following information on the spectrum line:

  • Spectrum type (1 and data type of channel e.g. long).
  • Gate that is applied (if any).
  • Range of parameter covered by the X-axis and number of bins on that axis
  • Where applicable the units of measure of the X-axis
  • If the spectrum is gated, the Gate column names the gate that is applied to the spectrum

The following additional information is available by drilling down into the spectrum object.

  • If the spectrum has a fold applied, the fold is listed with a gate icon as "Folded on : gatename", where gatename is the name of the gate used to fold the spectrum.
  • The parameters the spectrum histograms.

2-d Gamma Spectra

Without drilling down into the object, the following information is available on the Spectrum object line of the browser:

  • Spectrum type and channel data type.
  • X axis description including the parameter range covered by the axis, the axis binning and a guess at the units taken from the first parameter in the spectrum parameter list
  • Y axis description including the parameter range covered by the axis, the axis binning and a guess at the units taken from the first parameter in the spectrum parameter list
  • If the spectrum is folded, the fold is listed with a gate icon and named: "Folded on : gatename" where gatename is the name of the gate used to fold the spectrum.
  • A list of the parameters that are histogrammed by the spectrum.

Chart recorder spectra

Without drilling down, the following information is available on a chart recorder spectrum line.

  • The Spectrum type and channel data type.
  • The "time" axis length and binning
  • If appropriate, the time axis units

By drilling down into a chart recorder spectrum, the following additional information is available

  • The parameter on the time axis named as: "Time : parameter name"
  • The parameter on the 'value' axis named as: "Value : parameter name"
