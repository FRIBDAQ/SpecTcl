The save state dialog

This dialog allows you to save the state of the program. In the top part of the dialog, select the elements of the state you want save. In the middle part of the dialog either select a file to overwrite or navigate to a directory and type in the name of the saved file. Finally in the bottom part click one of:

  • Ok - to create the save file
  • Cancel - To dismiss the dialog and do nothing
  • Help - To display this help text
