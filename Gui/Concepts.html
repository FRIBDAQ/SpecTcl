SpecTcl GUI Browser Concepts

The SpecTcl Gui revolves around a hierarchical browser of the SpecTcl Objects. The browser allows you both to drill down to any desired level of detail about the hierarchy as well as to directly manipulate elements of the hieararchy.

Each of the SpecTcl object types is stored in a top level folder in the hiearchy. Opening this folder reveals subfolders and objects. The objects themselves may be opened to show their detailed definition.

Direct manipulation is performed via context menus and selection actions. A context menu is a menu that is associated with a right click over an object. Top level folders and all objects have context menus associated with them that allow you to do the most common operations on the item. A selection action is a double click over an object (in general folders do not have selection actions. The selection action for an object usually will allow you to edit the definition of an object.
