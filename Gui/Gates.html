Context menus and selection actions for gates

If you com across terms that are not familiar to you, please have a look at the online glossary.

Gates are stored within the Gates top level folder. The top level folder has a context menu that is activated when ou right click the folder. Gates themselves have both a right click activated context menu and a double-click activated selection action.

Top level Gate folder context menu

Right clicking the top level Gates folder brings up a context menu that has the following commands:

  • New... Allows you to use the gui to create a new gate. Note that Some gates, slices, contours and bands, in particular, are probably easier to enter on the displayer than through the SpecTcl GUI.
  • Refresh TreeRefreshes the browser tree. Normally the tree will refresh itself as needed whenever changes are made. However if changes are made the GUI is not aware of, you may need to manually refresh the tree.

Gate object context menus

Gate objects have a context menu that is activated by right clicking on any gate object. This context menu h as the following commands:

  • Edit... Allows you to edit the gate.
  • Apply To...Allows you to apply the gate to one or more spectra
  • DeleteAllows you to delete the gate. Note that in reality, gates are not deleted, they are turned into false gates. This choice was made to reduce the ripple effect of deleting a gate that other gates depend on.
