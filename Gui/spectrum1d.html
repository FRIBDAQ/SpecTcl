1-d Spectrum editor

This is the editor for 1-d and bitmask spectra. A 1-d spectrum is defined on a single parameter. You must select a spectrum name, select a parameter and describe the parameter axis of the histogram. You may also optionally select a gate to apply to the spectrum. Applying a gate will only allow events that satisfy the gate to increment the spectrum. Note that if you dont' want to create a 1-d spectrum you can change the spectrum type by clicking the 1-d button and selecting a new spectrum type from the drop down menu.

At the top of the dialog we have:

  • An entry labelled Spectrum Name:; Type the name of the spectrum you are creating here.
  • Some text that will display any gate you may want to apply to the spectrum. the spectrum specific editors will give you the opportunity to supply a gate
  • An array checkbox. If you build a 1-d spectrum from a treeparameter array element, you can create an entire array of spectra, one on each element of the array if this is checked

In the middle of the dialog we have a limited object browser. This browser makes the parameters, and gates available.

  • Double clicking on a parameter selects it for use on the spectrum's parameter axis. If you accidently select the wrong parameter, simply double-click a different parameter. If the parameter you choose is a tree-parameter suggested axis limits and binning will be given, as well as the parameter units of measure. You can override the limits and binning by typing new values in the Low, High, and Bins entries. Note when creating a bitmask spectrum you will almost always have to modify the limits and binning. Usually Low = 0, Hihgh = 31, bins = 32 is correct.
  • Double clicking a gate applies it to the spectrum. If you select the wrong gate by mistake, double click a new gate. If you decide not to gate the spectrum, double click the gate name in the top part of the dialog.

The buttons that are avaialable at the bottom of the dialog are:

  • Ok Creates the new spectrum as described and dismisses the dialog
  • Accept Creates the new spectrum but leaves the dialog up to allow you to create more spectra.
  • Cancel Cancels spectrum creation and dismisses the dialog.
  • Help Brings up this help dialog
If you attempt to create a spectrum that already exists, you will be prompted to confirm. If you accept, the existing spectrum will be deleted before the new one is created.
