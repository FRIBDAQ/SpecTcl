Slice gates

Slice gates are easier to enter graphically!!!

You can enter slice gates using Xamine by displaying a spectrum with the appropriate parameter(s), Clicking on the Cut button in the Xamine window and clicking in points on the spectrum.

Assuming you still want to enter a slice gate using the nongraphical editor; The left side of the editor is an object browser limited to parameters. The right side a listbox of the parameters in the gate. For ordinary slices, only one parameter will be allowed in this listbox. For gamma slices, any number of parameters area allowed.

To add a parameter to the listbox, double click on it in the browser. If you have made an error, double click the parameter in the listbox to remove it. If the gate type is a slice, double clicking a parameter in the browser while a parameter is already loaded into the listbox removes the existing parameter from the listbox.

At any point you can enter the low and high limits of the gate in the two entries at the bottom of the editor.

The buttons at the bottom of the editor are:

  • Ok Click this to create the gate and dismiss the dialog.
  • Accept Click this to create the gate and leave the dialog up
  • Cancel Click this to dismiss the dialog without making a new gate
  • Help Clicking this displays this help message. If you have selected a different gate type, clicking help again will display the help for the specific gate editor you have selected.
