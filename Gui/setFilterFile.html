Set the Filter File

Filters need an output file and can be created either enabled or disabled. The output file is the file to which filtered events will be written. An enabled filter processes data and writes filtered events to an output file. Disabled filters do not process data and will not write to an output file.

The top part of this stage of the filter wizard allows you to select a filter output file. Warning if you chose an existing file, the file will be overwritten, rather than appended. The checkbox labelled Active allows you to determine if the filter starts out enabled or not. If the Active checkbox is checked, the filter will start out enabled.

The buttons at the bottom of the wizard have the following functions:

  • Cancel Dismisses the wizard without creating a new filter.
  • Help Displays this help message
  • Back Returns to the previous step of the filter wizard (selecting filter parameters>
  • Finish Creates the filter you have used the wizard to describe
