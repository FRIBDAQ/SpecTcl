SpecTcl objects and the browser

The SpecTcl browser manipulates the four major SpecTcl object types:

  • Spectra (or histograms if you prefer)
  • Parameters both tree and non tree parameters.
  • Gates
  • TreeVariables

Each of these object types is managed by the browser as a folder containing a hierarchy of objects. The hierarchy is constructed by using the period "." to represent separators of hierarchy levels. Thus, for example, the tree parameter named event.raw.de is represented as: 

                event +
                      +-->raw +
                              +-->de

Each object can be opened up to show increasing levels of detail about its definition. In addition right clicking top level folders and objects brings up a menu of actions that can be performed on the object, or objects of that type (in the case of the top level folders). Double clicking an object will also bring up an editor for the object. The remainder of this page will describe:

  • The information available for fully opened Spectra
  • The information available for fully opened Parameters
  • The information available for fully opened Gates
  • The information available for fully opened Variables
