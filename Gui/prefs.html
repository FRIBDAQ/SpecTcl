Preferences editor

The preferences editor allows you to edit some of the default values for the GUI. These defaults are stored in ~/.SpecTclDefaults, and look like an ordinary, editable text file.

At the bottom of the preferences editor are three buttons:

  • Ok - Accepts the changes, and writes them out to your preferences file ~/.SpecTclDefaults
  • Cancel - Dismisses the dialog, making no changes.
    • Help - Displays this help message
