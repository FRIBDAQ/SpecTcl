Gamma summary spectrum editor

A gamma summary spectrum is a spectrum that is very much like a summary spectrum. However, while a summary spectrum displays a simple 1-d spectrum in each vertical strip of the spectrum, the gamma summary spectrum displays a gamma 1d spectrum in each vertical strip of the spectrum.

To define a gamma summary spectrum, you must define the parameters for each vertical strip (g1) spectrum one strip at a time. When done with a strip, click the NextChannel button to move on to the next channel. Parameters can only be added to the last strip.

There are two ways to add a parameter to the current strip.

  1. Click a parameter in the Parameters folder of the browser window of the spectrum editor or
  2. Click a gamma spectrum in the Specrum folder of the browser window of the spectrum editor to add all of the parameters that are histogrammed by that spectrum to the current channel list.

The entry boxes below the parameter lists listsbox allows you to define the Y axis definition for the spectrum. If any item is empty when you add a parameter to the spectrum, and the parameter added has a suggested axis range, the item will be filled in from the parameter definition.

Parameters can be removed from any channel by double clicking the parameter in the Parameter Lists list box. If removing a parameter would leave no more parameters in a channel, that channel is eliminated and all of he channels below it are re-numbered.

As with all spectrum editors, double clicking a gate in the Gates folder will set that gate as the initial gate on the spectrum.

The OK buttton at the bottom of the dialog will define the spectrum and dismiss the editor. The Accept button will define the spectrum and leave the editor displayed so that another spectrum can be defined. The Cancel button will dismiss the editor without defining a new spectrum. The Help button will display this help page.
