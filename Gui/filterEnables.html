Enabling filters

This dialog allows you to enable or disable filters. The dialog consists of two sections. A table of filters at the top of the dialog, and a set of buttons at the bottom of the dialog.

The filter table columns have the following meanings and purpose:

  • Enable contains a checkbox you can select to enable the filter or deselect to disable the filter
  • Name contains the name of the filter described by that row of the table
  • Gate contains the gate that determines if an event is written by the filter
  • Parameters Contains a list of the parameters that will be written to file whenever an event satisfies the filter's gate
  • State Contains the state of the filter as of when the dialog was brought up.

The buttons along the bottom of the dialog have the following function:

  • Ok Accept all of the changes you have made to the filter enables and dismiss the dialog.
  • Cancel dismiss the dialog without making any changes
  • Help displays this help text
